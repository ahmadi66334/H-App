## 🧱 Inside an H-App

An H-App is a plain HTML5 document that contains:

- Multiple pages (hidden/shown dynamically)
- A complete set of UI components (text, titles, images, buttons, cards, forms, products, blog posts, videos, webviews, counters, progress bars, alerts, accordions, custom blocks)
- Navigation systems (bottom menu, top toolbar, drawer menu)
- A customizable loader screen
- Built‑in JavaScript for navigation, data storage (localStorage), games, and forms
- Optional meta tag (`happ-meta`) for integrity verification

Despite being a single HTML file, it behaves exactly like a native app.
